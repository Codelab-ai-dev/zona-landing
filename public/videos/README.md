# Hero background video

Coloca aquí el archivo `hero.mp4` exportado de tu video (por ejemplo, el que compartiste como `VIDEO-2025-11-03-19-49-25.mp4`).

Asegúrate de comprimirlo para web (idealmente menos de 5 MB, 1080p máximo, `muted` y `loop`) y renómbralo a `hero.mp4` para que el componente Hero lo cargue automáticamente.
